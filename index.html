02 — Contexto

De onde vem a ideia

🏦

Experiência em Fintech

Na BMP, disponibilizávamos serviços bancários completos via API, de onboarding a transações. Porém, muitos dos nossos parceiros não conseguiam implementar a validação biométrica facial: o prazo de desenvolvimento era curto, as equipes eram enxutas e a integração de biometria era tratada como algo não importante.

Percebi que o gargalo não era a tecnologia, era a complexidade de integração. Se o processo fosse 100% via API, qualquer equipe poderia implementar em dias.

⚡

O Problema que estou resolvendo

Integrar biometria facial costuma exigir meses de desenvolvimento e equipes especializadas. O KYC TrueFace entrega tudo via API + Webhook, reduzindo drasticamente o custo e o tempo.

~60%
Redução no tempo de integração
8–10
Dias para integrar via API

03 — Solução

Como o KYC TrueFace funciona

01

Envio de Imagens

O cliente envia documento de identidade + selfie via chamada de API REST

02

Processamento Biométrico

AWS Rekognition compara o rosto do documento com a selfie enviada

03

Fila de Mensageria

AWS SNS + SQS gerenciam o fluxo assíncrono com garantia de entrega

04

Webhook de Resultado

O parceiro recebe o resultado (aprovado / pendente / reprovado) em seu endpoint

05

Consulta de Status

API de consulta disponível para verificar o resultado a qualquer momento

🔑 100% via API — sem SDK proprietário, sem dependência de plataforma. Qualquer linguagem, qualquer stack.

📋 Revisão manual integrada — casos mais sensíveis são enviados para análise no painel administrativo.

🔔 Webhook confiável — entrega garantida com retry automático via SQS.
